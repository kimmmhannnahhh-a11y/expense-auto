# 법카 영수증 자동등록 (다우오피스 works > 지출관리)

폰/PC에서 영수증 사진을 찍으면 AI가 금액·날짜·가게를 읽어 지출항목·제목을 추천하고,
다우오피스 works > 지출관리에 사진첨부+항목선택+제목입력+등록까지 자동으로 해주는 도구.

## 작동 흐름
1. 웹 열기(폰/PC) → 영수증 사진 촬영/선택
2. AI 자동 분석 → 금액·날짜·가게·지출항목·제목 자동 채움 (수정 가능)
3. "다우오피스에 등록" → 서버가 자동 로그인 후 등록

## 2단계 구성
- **1단계 (완료):** 사진 → AI 인식 → 항목·제목 추천 화면. ANTHROPIC_API_KEY만 있으면 작동.
- **2단계 (진행중):** 다우오피스 자동 로그인·입력·등록. 실제 페이지 보고 `config.js`의 SELECTORS와 CATEGORIES를 채우면 작동.

## 설정 (Render 환경변수)
`.env.example` 참고. ANTHROPIC_API_KEY / DAOU_LOGIN_URL / DAOU_ID / DAOU_PW.
비밀번호는 코드에 넣지 말고 반드시 환경변수로.

## 직접 채워야 할 곳 (다우오피스 화면 확인 후)
- `config.js` → `CATEGORIES`: 실제 지출항목 목록
- `config.js` → `SELECTORS`: 다우오피스 로그인/지출관리 버튼·입력칸 위치
